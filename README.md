# Rogue Trader Build Wiki

정적 HTML로 보는 **Warhammer 40,000: Rogue Trader** 빌드 검색 도구입니다.

이 저장소의 빌드는 제가 직접 만든 빌드가 아닙니다. 빌드 데이터는 Revan619 커뮤니티에서 관리하는 빌드 스프레드시트를 검색하기 쉽게 정리한 것입니다. 원본 빌드 제작자와 커뮤니티 작업에 크레딧이 있습니다.

원본 스프레드시트:
[Revan619's Community Rogue Trader Unfair Builds & Resources](https://docs.google.com/spreadsheets/d/1rskX4sYcNm6Wqt4rtm8EQqRR4__yrEuxCEzjwoKlHOY/htmlview?pru=AAABnz4CWi4*jNqH2wVz4fMgnmJ2v2y72A)

## 사용

브라우저에서 `index.html`을 열면 됩니다.

- 캐릭터별 빌드 선택
- 레벨업 표 검색
- 현재 사용 중인 빌드 등록
- 빌드별 아이템 표
- 아이템 위치 확인
- 한글패치 기준 한글명 표시

## 데이터 기준

현재 페이지는 Revan619 스프레드시트의 DLC 3 빌드 시트를 기준으로 만든 정적 결과물입니다.

- `Companion Builds (DLC 3)`
- `Rogue Trader Builds (DLC 3)`

한글명은 로컬에서 한글패치가 적용된 게임 언어 파일과 대조해 생성했습니다. 패치 버전이나 번역 상태에 따라 일부 표기는 게임 내 최신 표기와 다를 수 있습니다.

## 포함/제외

저장소에는 브라우저에서 바로 쓰기 위한 정적 페이지와 필요한 이미지 자산만 포함합니다.

대용량 생성 JSON, 원본/출력 XLSX, 로컬 검사 로그, 추출 작업 폴더는 `.gitignore`로 제외했습니다.

## 크레딧

- Build data: Revan619 and the Rogue Trader community spreadsheet contributors
- Viewer/search UI: local static viewer for easier lookup
